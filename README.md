# fsp-finals-backend

## How to

1. Create `.env` with `POSTGRES_PASSWORD`
2. Add your `openai_key` to `config/setting.json`
3. Install deps with `rye sync`
4. Build images with `sh init.sh`
5. Spin everything up with `docker-compose up`
